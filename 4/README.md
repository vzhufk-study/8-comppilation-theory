### Тема: Елементи компіляції

Завдання

Ознайомитись з інтерпретатором для мови SPL.

1.  Створити і налагодити програму на мові SPL. Для реалізації програми створити декілька функцій. Бажано використовувати глобальні змінні, константи, передачу параметрів функціям, можливо, і функції main (4 бали).

2.  Провести лексичний аналіз створеної SPL-програми (побудувати таблиці ідентифікаторів і лексем) (4 бали).

3.  Провеcти граматичний розбір (за правилами граматики) методом рекурсивного спуску фрагменту створеної SPL-програми (4 бали).

4.  Проаналізувати об’єктну програму (побудувати стек із записами активації для кожної функції та поставити у відповідність операторам створеної SPL-програми фрагменти проміжного коду) (4 бали).

5.  Розібрати коди програм (давати відповіді на питання викладача по фунціях), які винесені в мережу (4 бали)

* лексичний аналіз (файл lexan.cpp);

* синтаксичний аналіз (syntax. cpp);

* інтерпретатор (inter.cpp);

* робота з таблицями (table.cpp);

* типи даних і прототипи функцій (protot.h).

Зауваження. Пункти 1 і 5 можна виконати спрощеним способом (по 2 бали - за кожний такий пункт):

1.  Розробити програму мовою SPLU (2 бали)

2.  Побудувати таблицю функцій для створеної програми та таблиці об'єктів для кожної функції (2 бали)

За умови виконання пункту 1 спрощеними способом, пункти 2-4 оцінюються, як раніше, по 4 бали, але виконуються для програми, створеною мовою SPLU
